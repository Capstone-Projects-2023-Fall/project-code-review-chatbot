# Introduction
Tired of teammates responding with "LGTM" without actually reading your code? Do you want real feedback of your code? With CodeReviewChatBot you can now have real time feedback and critique of code without another person. The chatbot has an interactive chat to allow users engage in a conversation about the bot's feedback. Elevate your code by interacting with the chatbot to receive effective and informative code review!
